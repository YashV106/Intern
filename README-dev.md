## Run on localhost (single command)

### 1) Install root dev dependency
From the repo root:
- `npm install`

### 2) Start everything
- `npm run dev:all`

### Expected URLs
- Backend (Express): http://localhost:5000
- Frontend (Next.js): http://localhost:3000

## Notes
- Backend env is in: `internshala-clone-main/backend/.env` (it already exists in the folder).
